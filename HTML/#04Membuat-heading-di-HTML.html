Belajar Heading di HTML

Heading di HTML ada enam, yakni H1, H2, H3, H4, H5, dan H6. Heading berfungsi untuk membuat judul unutk artikel dan juga sub judul pada bagian artikel.

Membuat Sub Judul

Sub judul atau sub heading dibukai dengan tag H2. Lalu diikuti dengan tag selanjutnya hingga sampai H6. Sementara itu, tah H1 hanya digunakan untuk judul artikel saja.

Heading dengan huruf besar semua

Heading adalah judul sebuah artikel dan bagian dari artikrl. Ada enam level heading pada HTML, yakni dumulai dari H1 sampai H6.

Hedaing dengan huruf bear didepan

Properti text-transform: capitalize; berfungsi menentukan style case dan dari text. Properti ini bisa diberikan nilai capitalize, upperscale, lowercase, initial, none, inherti.

Hedaing dengan warna Merah

Atribut color adalah actirbut CSS yang berfungsi untuk memeberikan warna pada teks. Pada contoh ini, teks pada heading akan bewarna merah, karena kita memberikan warna red untuk color.

Heading dengan Latar Kuning

Atribut background-color adalah atribut untuk memberikan warna latar(background) pada elemen tertentu. Pada contoh ini, kita memebrikan latar kuning untuk heading.
